## How to view it locally

No server needed. Just open `index.html` in Chrome:

1. Download or clone this repository
2. Find the `index.html` file in the `designer-who-codes` folder
3. Drag it into Chrome (or double-click it)
4. Done — the whole course works from your Mac, no internet required (except for Google Fonts)

## How to host it free on GitHub Pages

This is also what you'll learn in Chapter 05. Here's the quick version:

### Step 1 — Create a GitHub account
Go to [github.com](https://github.com) and sign up (free).

### Step 2 — Create a new repository
- Click the **+** icon → **New repository**
- Name it: `designer-who-codes`
- Set it to **Public**
- Click **Create repository**

### Step 3 — Upload the files
Option A — Using GitHub Desktop (recommended for beginners):
1. Download [GitHub Desktop](https://desktop.github.com) (free)
2. Sign in with your GitHub account
3. Clone the repo you just created to your computer
4. Copy all the files from this folder into the cloned repo folder
5. In GitHub Desktop: write a commit message ("Add course files") → Commit → Push

Option B — Upload directly on github.com:
1. Open your repo on github.com
2. Click **Add file** → **Upload files**
3. Drag all the files and folders (styles/, scripts/, chapters/, index.html, README.md)
4. Click **Commit changes**

### Step 4 — Enable GitHub Pages
1. In your repo on github.com, click **Settings**
2. Scroll down to **Pages** in the left sidebar
3. Under **Source**, select **Deploy from a branch**
4. Under **Branch**, select **main** → **/ (root)** → click **Save**
5. Wait 1–2 minutes

### Step 5 — Your site is live!
Go to: `https://[your-github-username].github.io/designer-who-codes/`

Share that link. That's your course, live on the internet, for free.
